# Sparks App

 ## this is a simple Android app built using kotlin.

### The user interface was designed to be:
- *Simple and user-friendly*: Easy for users to input data and understand results.
- *Visually engaging*: A gradient background and centered layout improve appearance.
- *Accessible*: Clear labels and readable text ensure usability.
- *Interactive*: Buttons provide immediate feedback and actions.

### Key UI Elements:
- *EditText*: Allows user to input the time of day.
- *TextView*: Displays the social suggestion or error message.
- *Buttons*:
  - "Enter" button to process input
  - "Reset" button to clear input and output

## App Features

- User enters time of the day (e.g. Morning, afternoon , Dinner, etc.)
- the app suggests a social interaction ("social spark")
- Handles invalid input with helpful error messages
- includes a reset button

## Functionality

The app uses conditional logic (if statements) to determine which suggestion to display.

### Supported Inputs:
- Morning
- Mid-morning
- Afternoon
- Afternoon Snack Time
- Dinner
- Night / After Dinner

### Example:
If the user enters *"Mid-morning"*, the app suggests:
"Reach out to a collegue with a quick 'thank you' "

##  Error Handling

The app includes input validation to improve user experience:
- Displays an error if the input field is empty
- Accepts different formats (e.g., "Mid-morning", "mid morning")
- Shows helpful and friendly error messages
- Highlights incorrect input using setError()
